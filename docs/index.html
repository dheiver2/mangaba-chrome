mangaba.ai

A Brazilian-made AI assistant right in your browser. Chat about the current page and put AI agents to work browsing, researching, filling out forms, and interacting on social media for you.

Manifest V3 · zero dependencies · 100% local

What it does

💬 Chat with the page

Reads the active tab's content and answers questions about it, with Markdown replies and live streaming.

🤖 Agent mode

A team of agents performs real tasks in the browser: navigating, clicking, typing, scrolling, and reading — deciding step by step.

📝 Forms

Maps fields with labels and options, fills them in batches, validates against the site itself, and never invents personal data.

🔎 Web search

Searches across multiple sources, cross-references the information, and answers citing the domains it used.

💬 Social media

Reads inbox and DMs, drafts replies, likes and comments — with your confirmation before publishing anything.

🔐 Secure login

Opens the login screen and fills in the username, but never types your password — you finish signing in.

👁️ Vision

Takes a screenshot and describes it with a vision model when the page text isn't enough.

⚡ Fast

Multiple actions per step, KV-cache, model pre-warming, and DOM stabilization (browser-use-style techniques).

🔒 Secure

HTML escaped against XSS, human confirmation on sensitive actions, and defenses against prompt injection from pages.

How to install

  1. Download the ZIP and unzip it into a folder.
  2. Open chrome://extensions in Chrome (or Edge).
  3. Turn on Developer mode (top-right corner).
  4. Click Load unpacked and select the unzipped folder.
  5. Click the Mangaba icon to open the side panel.
  6. In ⚙︎, set your Mangaba Gateway URL (default http://localhost:8080/v1/chat/completions) and model.
⚙️ The extension needs a Mangaba Gateway (OpenAI-compatible API) to work. For agent mode, prefer larger models (e.g. mangaba-pro / mangaba-max) — smaller ones make more mistakes on the steps.
